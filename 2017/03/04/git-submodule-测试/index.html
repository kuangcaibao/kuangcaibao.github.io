2. 测试 git submodule 工作现象

2.1 裸仓库和一般仓库的区别

裸仓库和一般的 git 仓库有什么区别？主要有下面2点：

  1. 裸仓库没有工作区，一般 git 仓库有工作区

  2. 执行 git push 命令，可以推送修改到裸仓库，而一般的仓库不能被推送
